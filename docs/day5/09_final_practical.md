# Integrated Capstone Practical: River Basin Delineation and Analysis

This capstone laboratory integrates all skills learned throughout the 5-day training program.

---

## 1. Integrated Workflow Tasks
In this capstone, participants will execute the following steps:

1. **Setup Project Directory:** Create a standardized database folder.

2. **Terrain Conditioning:** Import a raw DEM, run SAGA **Fill Sinks**, and delineate the study watershed boundary polygon.

3. **Multispectral Classification:** Import Sentinel-2 bands, calculate NDWI, reclassify, and convert the output to vector water body polygons.

4. **Symbology & Style:** Apply graduated styles to terrain and rivers, compile an $A3$ map layout in QGIS, and export the layout as a print-ready PDF.
