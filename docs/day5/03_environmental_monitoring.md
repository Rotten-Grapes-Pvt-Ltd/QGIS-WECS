# Environmental Monitoring Applications

Geospatial indices are combined with terrain derivatives to monitor basin health and drought occurrences.

---

## 1. Drought Monitoring and Vulnerability

* **Vegetation Condition Index (VCI):** Normalizes NDVI values over a long-term historic record to evaluate drought severity.

* **Soil Water Index (SWI):** Tracks root-zone soil moisture anomalies using satellite microwave backscatter data, indicating agricultural drought onset.

---

## 2. Land Degradation Assessment

* By overlaying slopes, soil types, and NDVI trends, planners can map land degradation and prioritize watersheds for restoration.
