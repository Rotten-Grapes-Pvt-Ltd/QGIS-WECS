# QGIS Plugins and Extensions

QGIS features a modular structure, allowing users to extend its capabilities by installing community-developed plugins.

---

## 1. Core Plugins

* **QuickMapServices:** Allows adding satellite and topographic basemaps (Google Maps, OpenStreetMap, ESRI) to your QGIS map canvas.

* **Profile Tool:** Generates cross-sectional elevation profiles along a drawn line or vector river track.

* **Semi-Automatic Classification Plugin (SCP):** Tools for downloading and classifying optical and radar satellite imagery.
