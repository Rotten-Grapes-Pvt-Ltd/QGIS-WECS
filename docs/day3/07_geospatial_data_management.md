# Geospatial Data Management

Effective data management ensures that spatial datasets remain organized, documentable, and accessible across project lifetimes.

---

## 1. Standard Project Folders
Before starting any spatial project, enforce this directory layout:

* `/data/vector/`: Shapefiles, Geopackages, GeoJSONs.

* `/data/raster/`: DEMs, satellite bands, precipitation grids.

* `/projects/`: QGIS workspace files (`.qgz`).

* `/outputs/`: Exported layouts, tabular reports.

---

## 2. Metadata Standards
Metadata is "data about data." It should detail:

* **Source:** Who collected the data?

* **Acquisition Date:** When was the data captured?

* **CRS:** What coordinate reference system is used?

* **License:** Are there reuse restrictions?
